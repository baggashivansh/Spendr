# Spendr

Spendr is a simple personal expense tracking application built using Java and Spring Boot.  
The project was inspired by a real problem. I often wrote notes during payments to remember where money was spent, but this became unstructured and difficult to track.

Spendr provides a structured way to log, store, and review expenses.

---

## Features

User registration and login  
Add expenses with amount, category, and date  
View expenses and monthly summaries  
Category-wise breakdown  

---

## Tech Stack

Java  
Spring Boot  
MySQL  
JPA and Hibernate
